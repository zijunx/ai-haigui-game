# AI 海龟汤游戏测试清单 (Test Checklist)

本清单用于验证游戏核心流程及最新优化项的完整性。

## 1. 游戏大厅 (Lobby)
- [ ] **视觉展示**：页面背景、标题、挑战卡片列表是否渲染正常（无白屏）。
- [ ] **数据加载**：卡片上是否正确显示剧本标题、难度等级及“汤面”摘要。
- [ ] **响应式验证**：在不同分辨率下（PC/Pad/Mobile）卡片排列是否美观（Grid 布局生效）。

## 2. 游戏进入与初始化 (Initialization)
- [ ] **点击进入**：点击剧本卡片是否正确跳转至 `/game/:id` 页面。
- [ ] **内容显示**：页面顶部是否显示当前剧本的标题，“汤面”浮窗是否默认展开且内容正确。
- [ ] **引导语**：AI 是否发送了第一条欢迎语，并包含了正确的故事汤面。

## 3. 核心游戏交互 (Chat Interface)
- [ ] **发送功能**：用户输入问题后，点击发送按钮或按下回车，消息是否立即显示在气泡中。
- [ ] **Loading 状态**：发送后，发送按钮和输入框是否被禁用，是否出现“AI 裁决中...”的动画提示。
- [ ] **AI 回复逻辑**：
    - [ ] **格式**：AI 的回答是否仅限于“是/不是/与此无关”。
    - [ ] **准确性**：AI 是否根据后端提供的秘密“汤底”进行了准确判定。
    - [ ] **打字机效果**：回复文字是否以逐字弹出的方式呈现。

## 4. 异常处理测试 (Error Handling)
- [ ] **网络异常**：断开后端或网络，提问后是否显示带有“⚠️”的红色错误提示。
- [ ] **重试机制**：点击错误消息下方的“点击重试”按钮，是否能自动重新发送上一次的问题。

## 5. 结局页面 (Result Page)
- [ ] **进入路径**：在游戏区域点击“查看结局”按钮，是否成功跳转至结算页。
- [ ] **戏剧化呈现**：进入页面后是否有“真相即将浮现...”的加载动画，随后汤底才揭晓。
- [ ] **字体大小**：确认汤底（Answer）部分的字体在桌面端和移动端是否大小协调（不再过大）。
- [ ] **对话回溯**：点击“回顾这场博弈”能否展开查看刚才所有的聊天记录。
- [ ] **操作按钮**：
    - [ ] **再来一局**：点击能否重新进入当前剧本并清空旧聊天记录。
    - [ ] **返回大厅**：点击能否回到首页。

## 6. 移动端适配专项 (Mobile Experience)
- [ ] **滚动功能**：
    - [ ] 游戏大厅：在手机上能否流畅上划查看下方更多剧本。
    - [ ] 结局页面：长文本汤底下方的内容（如按钮、脚注）能否通过上划露出。
- [ ] **输入框适配**：在手机上唤起键盘时，输入框是否被顶起（不被遮挡），是否使用了 `h-[100dvh]` 避免背景溢出。
- [ ] **点击手感**：所有按钮在触摸时的 `active:scale-95` 反馈是否灵敏，无 300ms 点击延迟。
